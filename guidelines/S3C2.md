# S3C2: Measurement Protocol Execution

## Brief
The harness executes the specified measurement protocol consistently across all evaluation instances, returning standardized scores with uncertainty estimates and supporting metadata.

## Essential Features

1. **Consistent Protocol Application**: Apply measurement protocols uniformly across all evaluation instances without variance
2. **Standardized Score Format**: Return scores in consistent, standardized format (numerical scores, categorical judgments, structured outputs)
3. **Uncertainty Estimation**: Compute and report uncertainty estimates (confidence intervals, standard errors) for all scores
4. **Supporting Metadata Capture**: Include metadata with scores (scoring method, parameters, timestamps, protocol version)
5. **Multi-Protocol Execution Support**: Execute multiple measurement protocols simultaneously and return aligned results
