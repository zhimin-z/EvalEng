# S3C6: Uncertainty Quantification

## Purpose
Compute uncertainty estimates for all primary scores per pre-specified analysis plan $\Sigma$ enabling robust model comparisons. Confidence interval construction via bootstrap resampling (percentile method, bias-corrected accelerated BCa, configurable iterations 1000-10000) or parametric methods when distributional assumptions justified (t-distribution for normally distributed metrics with assumption verification). Statistical significance testing: permutation tests (non-parametric comparison with exact p-values), t-tests (paired for within-subject, independent for between-subject), Wilcoxon signed-rank (non-parametric alternative), with pre-specified multiple comparison corrections (Bonferroni: $\alpha' = \alpha/k$, Benjamini-Hochberg FDR, Holm-Bonferroni). Effect size computation: Cohen's $d$ (standardized mean difference), Hedges' $g$ (bias-corrected for small samples), confidence intervals on effect sizes. Quantify multiple uncertainty sources separately: sampling variability (finite dataset size), measurement variability (rater disagreement, stochastic sampling), model variability (random seed differences). Decompose total uncertainty into components, propagate uncertainties through aggregations using Taylor approximation or Monte Carlo methods. Evaluate metrics across multiple random seeds capturing run-to-run variability. Report all metrics with explicit uncertainties: $\text{metric} = 0.85 \pm 0.03$ or $[0.82, 0.88]_{95\%}$. Distinguish statistical significance (p-value $< \alpha$) from practical significance (effect size $> \delta_{min}$). Enable robust model comparisons preventing spurious claims.

## Essential Capabilities
1. **Bootstrap confidence intervals** - Percentile method, BCa with bias correction, configurable coverage levels and iterations
2. **Parametric confidence intervals** - t-distribution CIs, Wilson score interval for proportions, with assumption verification (normality tests)
3. **Permutation testing** - Non-parametric significance testing with exact p-values
4. **T-tests implementation** - Paired t-tests (within-subject), independent t-tests (between-subject), with assumption verification
5. **Wilcoxon signed-rank tests** - Non-parametric alternative to paired t-tests
6. **Multiple comparison correction** - Bonferroni correction, Benjamini-Hochberg FDR, Holm-Bonferroni method
7. **Effect size computation** - Cohen's d, Hedges' g (bias-corrected), confidence intervals on effect sizes
8. **Sampling uncertainty quantification** - Estimated from finite sample size via bootstrap or analytical formulas
9. **Measurement uncertainty quantification** - Rater disagreement (human), stochastic sampling (LLM judge), repeated measurements
10. **Model variability quantification** - Random seed effects, initialization variability, via multiple runs
11. **Uncertainty decomposition** - Separates sampling, measurement, and model variability; reports components
12. **Uncertainty propagation** - Through aggregations and transformations (Taylor approximation or Monte Carlo)
13. **Multi-seed evaluation** - Runs across multiple random seeds; captures stochastic variability; reports aggregated statistics
14. **Significance vs practical significance** - Distinguishes statistical significance (p < α) from practical significance (effect size > threshold)
