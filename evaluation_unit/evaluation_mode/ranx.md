def paired_student_t_test(control, treatment, max_p=0.01):
    """
    Two-sided Paired Student's t-Test.
    Null hypothesis: system A (control) and system B (treatment) are identical
    """
    _, p_value = ttest_rel(control, treatment, alternative="two-sided")
    return p_value, p_value <= max_p
```
These tests compare metric scores across different retrieval systems through statistical analysis. They examine the distributions and differences in evaluation scores without executing any model-generated artifacts.

# From ranx/metrics/get_unjudged_lists.py
@njit()
def _get_unjudged_list(qrels, run, k):
    """Compute the unjudged lists (at k).
    Unjudged list is a binary list whose element in position p is 1 
    if the document retrieved in position p is unjudged, 0 otherwise.
    """
    k = fix_k(k, run)
    unjudged_list = np.full(k, 1.0)
    # Pattern matching to find unjudged documents
    for i in range(k):
        for j in range(qrels.shape[0]):
            if run[i, 0] == qrels[j, 0]:
                unjudged_list[i] = 0.0
                break
    return unjudged_list
```
This module identifies unjudged documents in retrieval results by pattern matching document IDs between runs and qrels. It performs structural analysis of the ranked lists without executing any artifacts.
